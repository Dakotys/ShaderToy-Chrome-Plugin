Shadertoy extension

With <3 by Patu[Futuris^AberrationCreations]

Click on Shader, then...

    • Resolution dividerALT + 1..9
    • ForwardALT + Arrow Right
    • RewindALT + Arrow Left
    • Pause timeALT + Arrow Up
    • Restart timeALT + Arrow Down
    • Fullscreen editSHIFT + CTRL + Enter
    • Take HQ screenshotSHIFT + CTRL + S
    • RecordSHIFT + CTRL + R

    On OSX use CMD instead of ALT.
